## How to Use

1. Replace `mattress1.jpg` and `mattress2.jpg` with your actual mattress images
2. Update the company name, prices, and product descriptions as needed
3. Customize the color scheme by modifying the CSS variables in `styles.css`
4. Test on different devices to ensure responsiveness

## Adding Mattress Images

To add the mattress images:
1. Save your mattress images as `mattress1.jpg` and `mattress2.jpg` in the same directory as `index.html`
2. Images should be at least 400x300 pixels for best results
3. Alternatively, you can use image URLs and update the `src` attributes in the HTML

## CSS Variables (Easy Customization)

Edit these in `styles.css` to change colors:
```css
--primary-color: #2c3e50;      /* Dark blue */
--secondary-color: #3498db;    /* Light blue */
--accent-color: #e74c3c;       /* Red */
--light-bg: #ecf0f1;           /* Light gray */
```
